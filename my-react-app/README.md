# PopX (React + Vite)

Pixel-perfect mobile UI centered on the page, based on the provided Adobe XD screenshots.

## Run locally

```bash
npm i
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Pages

- `/` Landing
- `/login` Login
- `/signup` Signup
- `/profile` Profile (protected)

## Hosting

- **Vercel**: SPA rewrite via `vercel.json`
- **Netlify**: SPA redirects via `netlify.toml`
